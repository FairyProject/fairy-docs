# Basic

Fairy provides a set of libraries to help you develop your plugins.
These libraries are designed to make your life easier by providing you with a set of tools and utilities that you can use in your plugins.

---
## Adventure
![adventure](img/adventure.png)

We've included the [Adventure](https://docs.advntr.dev/) library in Fairy to help you with text formatting and component building.
[MCPlayer](https://github.com/FairyProject/fairy/blob/v0.7/framework/platforms/mc-platform/src/main/java/io/fairyproject/mc/MCPlayer.java) is an instance of Audience, which is a part of the Adventure library.

---
## PacketEvents
![packetevents](img/packetevents.webp)

We've also included the [PacketEvents](https://github.com/retrooper/packetevents) library in Fairy to help you with packet manipulation.
You can easily listen to packets and modify them using the PacketEvents library.

---

With all these libraries included in Fairy, you can easily develop your plugins without worrying about the low-level details!
